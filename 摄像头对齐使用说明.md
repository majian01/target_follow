# 摄像头中点对齐跟随系统

## 概述

这个系统实现了机器人摄像头中点与检测目标中心的精确对齐，通过改进的控制算法确保目标始终保持在图像中心位置。

## 主要特性

✅ **精确对齐**: 使用比例控制算法实现像素级精度对齐  
✅ **平滑控制**: 集成转向平滑算法，避免震荡  
✅ **多种配置**: 提供三种预设配置适应不同场景  
✅ **实时监控**: 显示对齐状态、成功率和质量指标  
✅ **智能降速**: 对齐过程中自动调整速度  

## 文件说明

### 核心文件
- `cam_follow_final.py` - 最终版本的跟随节点（推荐使用）

## 关键改进点

### 1. 精确对齐算法
```python
def calculate_precise_alignment_turn(self, target_center_x):
    # 计算目标中心与图像中心的偏差
    x_error = self.config.get_alignment_error(target_center_x, self.mid_x)
    
    # 如果已经对齐，不转向
    if self.config.is_aligned(target_center_x, self.mid_x):
        return 0
    
    # 使用比例控制计算转向速度
    turn_speed = x_error * self.config.TURN_KP
```

### 2. 转向死区缩小
- 原始代码死区: ±256像素（80%图像宽度）
- 改进后死区: ±5像素（可配置）
- 提升对齐精度 **50倍以上**

### 3. 平滑控制
```python
# 平滑转向命令，避免震荡
smoothed_turn = (self.config.TURN_SMOOTH_FACTOR * self.last_turn_command + 
                (1 - self.config.TURN_SMOOTH_FACTOR) * turn_speed)
```

### 4. 智能速度控制
- 偏离中心时自动降速
- 提高对齐过程的稳定性
- 防止高速运动时错过目标

## 参数调优指南

### 如果对齐精度不够
1. 减小 `ALIGN_TOLERANCE`（对齐容差）
2. 增加 `TURN_KP`（转向系数）
3. 增加 `TURN_SMOOTH_FACTOR`（平滑因子）

### 如果出现震荡
1. 减小 `TURN_KP`（转向系数）
2. 增加 `TURN_SMOOTH_FACTOR`（平滑因子）
3. 减小 `MAX_TURN_SPEED`（最大转向速度）

### 如果响应太慢
1. 增加 `TURN_KP`（转向系数）
2. 减小 `TURN_SMOOTH_FACTOR`（平滑因子）
3. 增加 `MAX_TURN_SPEED`（最大转向速度）

## 自定义配置

编辑 `alignment_config.py` 文件中的 `AlignmentConfig` 类：

```python
class AlignmentConfig:
    ALIGN_TOLERANCE = 5      # 对齐容差 (像素)
    TURN_KP = 0.004         # 转向比例控制系数
    MAX_TURN_SPEED = 0.6    # 最大转向速度 (rad/s)
    TURN_SMOOTH_FACTOR = 0.7 # 转向平滑因子 (0-1)
```

## 故障排除

### 常见问题

1. **目标检测不到**
   - 确认YOLO节点正常运行
   - 检查 `/yolo/results` 话题是否有数据

2. **对齐效果不佳**
   - 尝试不同的预设配置
   - 根据调优指南调整参数

3. **机器人震荡**
   - 使用 `precise` 配置
   - 增加平滑因子
   - 减小转向系数

4. **响应太慢**
   - 使用 `fast` 配置
   - 适当增加转向系数

### 调试技巧

1. 观察日志输出的对齐状态
2. 关注对齐成功率和平均误差
3. 根据实际表现调整配置

## 技术细节

### 控制算法
- **比例控制**: P控制器实现精确对齐
- **死区控制**: 避免在目标区域内无意义转向
- **限幅控制**: 防止转向速度过大
- **平滑滤波**: 减少控制信号噪声
